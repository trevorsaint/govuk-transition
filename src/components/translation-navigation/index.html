Components Translation navigation

Displays a list of links to available translations.

GOV.UK component guide

Examples

Example 1

{{ appTranslationNavigation({ items: [{ href: '#', text: 'English', locale: 'en', active: true }, { href: '#', text: 'Cymraeg', locale: 'cy' }] }) }}

Example 2

Multiple translations

{{ appTranslationNavigation({ items: [{ href: '#', text: 'English', locale: 'en', active: true }, { href: '#', text: 'Français', locale: 'fr' }, { href: '#', text: 'हिंदी', locale: 'hi' }, { href: '#', text: '日本語', locale: 'ja' }, { href: '#', text: 'اردو', locale: 'ur' }, { href: '#', text: '中文', locale: 'zh' }] }) }}

Example 3

With data attributes

{{ appTranslationNavigation({ items: [{ href: '#', text: 'English', locale: 'en', attributes: { track_category: "categoryEnglish", track_action: 1.1, track_label: "labelEnglish", track_options: '{dimension28: 1, dimension29: "dimension29English"}' }, active: true }, { href: '#', text: 'Cymraeg', attributes: { track_category: "categoryWelsh", track_action: 1.2, track_label: "labelWelsh", track_options: '{dimension28: 1, dimension29: "dimension29Welsh"}' }, locale: 'cy' }], attributes: { 'aria-label': 'Translations', 'role': 'navigation' } }) }}

Example 4

Inverse

{{ appTranslationNavigation({ inverse: true, items: [{ href: '#', text: 'English', locale: 'en', active: true }, { href: '#', text: 'Cymraeg', locale: 'cy' }] }) }}

Example 5

With a brand colour

{{ appTranslationNavigation({ brand: 'wales-office', items: [{ href: '#', text: 'English', locale: 'en', active: true }, { href: '#', text: 'Cymraeg', locale: 'cy' }] }) }}

Example 6

Right to left (using class direction-rtl on the container)

{{ appTranslationNavigation({ items: [{ href: '#', text: 'English', locale: 'en', active: true }, { href: '#', text: 'Cymraeg', locale: 'cy' }] }) }}
