Components Published dates

Displays dates to reflect when content was published and updated.

GOV.UK component guide

Examples

Example 1

Displayed above the content

{{ appPublishedDates({ published: { timestamp: '1990-01-01T15:42:37.000+00:00' }, lastUpdated: { timestamp: '2016-10-20T12:15:42.000+00:00' } }) }}

Example 2

Displayed below the content

{{ appPublishedDates({ attributes: { id: 'published-dates-toggle-2', 'data-module': 'toggle', lang: 'en' }, published: { text: 'Published', timestamp: '1990-01-01T15:42:37.000+00:00' }, lastUpdated: { text: 'Last updated', timestamp: '2016-10-20T12:15:42.000+00:00' }, history: [{ timestamp: '1990-01-01T15:42:37.000+00:00', text: 'First published' }, { timestamp: '1995-07-20T15:42:37.000+0000', text: 'Updated to include information for 1994' }, { timestamp: '2000-10-14T15:42:37.000+0000', text: 'Updated information on pupil premium reviews and what information schools need to publish on their websites.' }] }) }}

Example 3

Above and below content

{{ appPublishedDates({ attributes: { id: 'published-dates-toggle-3a', 'data-module': 'toggle', lang: 'en' }, published: { text: 'Published', timestamp: '1990-01-01T15:42:37.000+00:00' }, lastUpdated: { text: 'Last updated', timestamp: '2016-10-20T12:15:42.000+00:00' }, linkToHistory: { href: '#published-dates-toggle-3b', text: 'see all updates' } }) }}
